# Tugas Pemrograman Web 1

Repository tugas mata kuliah Pemrograman Web 1.

## Struktur

| Folder | Deskripsi |
|--------|-----------|
| Pertemuan11 | Aplikasi absensi mahasiswa |
| pertemuan13 | Materi JavaScript (function, if-else, switch-case) |
| pertemuan14 | Materi JavaScript (array, array object) |
| pertemuan16 | CRUD Mahasiswa (PHP + MySQL) |

## Cara Menjalankan

1. Jalankan XAMPP (Apache + MySQL)
2. Import `pertemuan16/crud-mahasiswa/database.sql` ke phpMyAdmin
3. Akses `http://localhost/Tugas%20Pemograman%20Web%201/pertemuan16/crud-mahasiswa/login.php`
4. Login: `admin` / `admin`
