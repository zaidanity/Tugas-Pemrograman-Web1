# Tugas: Form Input Mahasiswa

* **Form Input:** NIM, Nama, Jurusan, dan Fakultas.
* **Tombol Aksi Utama:** Tambah dan Update.
* **Aksi per Record:** Fitur Edit dan Hapus pada setiap data yang ditampilkan.
* **Struktur Data:** Data dikelola menggunakan *array of objects*.
* **Persistensi:** Data wajib diintegrasikan dan disimpan ke dalam *Local Storage* browser.
